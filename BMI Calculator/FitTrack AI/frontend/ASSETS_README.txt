This folder is intended for frontend assets (icons, sounds, SVGs) used by the Tkinter app.

Place icon files (e.g., icon.ico) or sound clips here and update `backend/app.py` to use them.

Example:
 - icon.ico  -> set with `root.iconbitmap("path/to/icon.ico")`
 - click.wav -> play with winsound.PlaySound when on Windows
